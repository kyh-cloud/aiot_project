# IoT센서와 인공지능을 이용한 이상징후 차단 보안관리 시스템
 
최신 기술이자 화두인 IoT와 인공지능을 융합하여 더욱 정확하고 신속한 불법 침입 그리고 이상징후 차단 보안 관리 시스템. 각종 센서를 통해 화재 징후를 포착하고 이미지 센싱을 통하여 영상을 받아와 데이터 압축률을 높여 장시간 영상 습득, 장기간 보관을 목표로 함.
